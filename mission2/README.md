# Mission 2: Git과 함께하는 Python 기초

## 📝 목표
객체지향(OOP) 방식의 퀴즈 게임을 개발하고 JSON 데이터를 관리합니다.

## 📋 요구사항
- Quiz 클래스: 문제 및 선택지 관리
- QuizGame 클래스: 게임 진행 및 데이터 저장
- JSON 파일 (state.json)을 통한 최고 점수 저장
- Git을 통한 버전 관리

## 🚀 실행 방법
```bash
python main.py
```

## 📂 파일 구조
- `main.py`: 퀴즈 게임 메인 파일
- `state.json`: 게임 상태 저장 파일 (자동 생성)
