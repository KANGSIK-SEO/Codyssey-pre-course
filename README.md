1. **[Mission 1] AI/SW 개발 워크스테이션 구축**
   - Docker를 활용한 독립된 개발 환경 설정 및 컨테이너 관리.

   - 객체지향(OOP) 방식의 퀴즈 게임 개발 및 JSON 데이터 관리.
3. **[Mission 3] Mini NPU Simulator 개발**
   - 라이브러리 없는 순수 Python 반복문을 활용한 MAC 연산 및 성능 분석.
